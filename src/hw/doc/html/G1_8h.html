Detailed Description

4D SYSTEMS uDRIVE-uSD-G1 storage device control interface.

Complete interface to manage the 4D SYSTEMS uDRIVE-uSD-G1 embedded "DOS micro-Drive" device.

See 4D SYSTEMS uDRIVE-uSD-G1 datasheet and command set for more information.
