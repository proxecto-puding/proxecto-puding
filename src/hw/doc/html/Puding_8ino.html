short getChanterOffset ( )

Calculate the offset to apply to the base tone.

The chanter has a base tone defined and each fingering has an offset defined. The correspondent note is obtained by adding the offset to the base tone. Note that the offset can be negative.

Returns
An offset.

Check if the destination of the configuration data is this bagpipe.

Each Puding bagpipe has an unique serial product identificator number. This id is used to identify the bagpipe due to is possible to have more than one bagpipe connected to the same receiver.
