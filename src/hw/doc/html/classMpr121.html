unsigned int Mpr121::getTouchAndProximityInputs ( )

Read MPR121 touch and proximity input values.

Format: -|-|-|E12|E11|...|E0 (16 bits) E12: proximity input. E11-0: respective touch values. 1 = touched/proximate. 0 = untouched/not proximate.

Returns
All inputs in an unique 16 bits value. If there is a new value available (IRQ enabled) return this, else return the old value.
