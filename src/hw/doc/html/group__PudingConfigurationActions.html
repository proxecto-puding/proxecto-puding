Detailed Description

Puding configuration actions.

Different configuration actions allowed by the bagpipes.

ACTION_CURRENT refers to the current configuration. ACTION_NEW refers to a new configuration. ACTION_REVERT refers to undo the last changes and restore the previous configuration. ACTION_DEFAULT refers to undo the last changer and restore the default configuration.
